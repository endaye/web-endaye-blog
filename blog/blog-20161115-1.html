冒泡排序 Bubble sort

算法原理

冒泡排序是最简单的排序。 它重复地走访过要排序的数列，一次比较两个元素，如果他们的顺序错误就把他们交换过来。 走访数列的工作是重复地进行直到没有再需要交换，也就是说该数列已经排序完成。 这个算法的名字由来是因为越小的元素会经由交换慢慢“浮”到数列的顶端。

算法流程

  1. 一共N个元素arr[N]，从arr[0]到arr[n-1]。
  2. 两层for循环，外层i，内层j
  3. 比较相邻的元素。如果第一个比第二个大，就交换他们两个（降序反之）。
    swap(arr[j], arr[j+1])
  4. 对每一对相邻元素作同样的工作，从开始第一对到结尾的最后一对。
    在这一点，最后的元素应该会是最大的数。j = 0 to n-1-i
    这里最重要的就n-1-i，因为随着i增大，arr后面被排序好的大数就越多。
  5. 针对所有的元素重复以上的步骤，除了最后一个。i = 0 to n-2
  6. 持续每次对越来越少的元素重复上面的步骤，直到没有任何一对数字需要比较。

// Created by Yuancheng Zhang on 11/19/16.

// 方法一：（数组递增）
void BubbleSort(int * arr, const int N) {
    int i, j, temp;

    // arr[0], ... , arr[N-2], arr[N-2]
    // 永远保持最后一个最大
    for (i = 0; i < N - 1; ++i) {
        for (j = 0; j < N - 1 - i; ++j) {
            if (arr[j] > arr[j+1]) {
                temp = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = temp;
            }
        }
    }
}

// 方法二：优化之后（数组递增）
// i递减；
// corner cases check
// XOR swap
void BubbleSort(int * arr, const int N) {
    if(arr == 0 || N < 2) {
        return;
    }
    for (int i = N - 1; i > 0; --i) {
        for (int j = 0; j < i; ++j) {
            if (arr[j] > arr[j+1]) {
                arr[j] ^= arr[j+1];     // x ^= y
                arr[j+1] ^= arr[j];     // y ^= x
                arr[j] ^= arr[j+1];     // x ^= y
            }
        }
    }
}

选择排序 Selection sort

算法原理

选择排序的思想其实和 冒泡排序有点类似，都是在一次排序后把最小的元素放到最前面。 但是过程不同，冒泡排序是通过相邻的比较和交换。而选择排序是通过对整体的选择。

选择排序的主要优点与数据移动有关。如果某个元素位于正确的最终位置上，则它不会被移动。选择排序每次交换一对元素，它们当中至少有一个将被移到其最终位置上，因此对n个元素的序列进行排序总共进行至多n-1次交换。在所有的完全依靠交换去移动元素的排序方法中，选择排序属于非常好的一种。

算法流程

  1. 在未排序序列中找到最小（大）元素，存放到排序序列的起始位置。
  2. 再从剩余未排序元素中继续寻找最小（大）元素，然后放到已排序序列的末尾。
  3. 以此类推，直到所有元素均排序完毕。

// Created by Yuancheng Zhang on 11/20/16.

// 数组递增
void SelectionSort(int * arr, const int N) {
    // 边界检查
    if(arr == 0 || N == 0)
        return ;
    int min_index, min_value;

    // 遍历数组中的每一位
    for (int i = 0; i < N; ++i) {

        // 找到已排序序列的尾巴
        min_index = i;
        min_value = arr[i];

        // 找到后面的序列中中的最小值
        for (int j = i; j < N; ++j) {
            if (arr[j] < min_value) {
                min_index = j;
                min_value = arr[j];
            }
        }

        // 最小值与尾巴交换位置
        int temp = arr[i];
        arr[i] = arr[min_index];
        arr[min_index] = temp;
    }
}

插入排序 Insertion sort

算法原理

插入排序 是一种简单直观的排序算法。它的工作原理是通过构建有序序列，对于未排序数据，在已排序序列中从后向前扫描，找到相应位置并插入。插入排序在实现上，通常采用in-place排序（即只需用到O(1)的额外空间的排序），因而在从后向前扫描过程中，需要反复把已排序元素逐步向后挪位，为最新元素提供插入空间。

算法流程

插入排序和打扑克的抓牌的过程类似。
牌库的牌：未排序的序列
手中的牌：已排序的序列
抓牌入手：从未排序的序列中选择下一个元素，在已排序的序列查找相应位置，将其插入，其他相应的已排序元素向后移
  1. 从第一个元素开始，该元素可以认为已经被排序
  2. 取出下一个元素，在已经排序的元素序列中从后向前扫描
  3. 如果该元素（已排序）大于新元素，将该元素移到下一位置
  4. 重复步骤3，直到找到已排序的元素小于或者等于新元素的位置
  5. 将新元素插入到该位置后
  6. 重复步骤2~5

// Created by Yuancheng Zhang on 11/20/16.

// 数组递增
void InsertionSort::Sort(int * arr, const int N) {
    // 边界检查
    if (arr == 0 || N < 2) {
        return;
    }

    // 假设第一个数位置时正确的；要往后移，必须要假设第一个。
    for (int i = 1; i < N; ++i) {

        int j = i;
        int target = arr[i]; //待插入的

        // 从i开始，找到目标位置，其余后移
        while(j > 0 && target < arr[j-1]) {
            arr[j] = arr[j-1];
            --j;
        }

        // 插入
        arr[j] = target;
    }
}

C++ 语言实现


// Created by Yuancheng Zhang on 10/4/16.

#include "shell_sort.h"

/* shellsort: sort input_array[0]...intput_array[size-1] into increasing order */
void ShellSort::Sort(int * input_array, const int input_array_size) {
    int gap, i, j, temp;

    // 最外层的for语句控制两个被比较元素之间的距离，从n/2开始，逐渐进行对折，直到距离为0
    for (gap = input_array_size / 2; gap > 0; gap /= 2) {
        // 中间层的for循环用于在元素间移动位置
        for (i = gap; i < input_array_size; ++i) {
            /*
             * 最内层的for语句用于比价各对相距gap个位置对元素，当这两个元素逆序时把它们互关
             * 由于gap的值最终要递减到1，因此所有元素最终都会位于正确的排序位置上
             */
            for (j = i - gap;
                 j >= 0 && input_array[j] > input_array[j+gap];
                 j -= gap) {
                // switch [j] and [j+gap] values
                temp = input_array[j];
                input_array[j] = input_array[j+gap];
                input_array[j+gap] = temp;
            }
        }
    }
}
